# Project 3: Around The U.S.

**Intro**

This is the third project in the Practicum Software Engineer course. This was created using Html and CSS.
This project focused on implementing responsive web design.

### Project features

- Semantic HTML 5
- Flexbox and Grid
- Positioning
- Media queries
